# NES Emulator

A work-in-progress emulator for the Nintendo Entertainment System (NES) written in C++. 

## Roadmap

- ✅ CPU 
- ✅ BUS
- ✅ ROM
- ✅ PPU (need to fix glitches) 
- ✅ GamePad
- [ ] APU <- **currently working on**
